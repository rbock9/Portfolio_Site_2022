GameGallery

This Vue app displays user-generated information about various video games, and allows users to create, read, update, or delete (CRUD) data relating to their personal game collection. I created this during General Assembly's Software Engineering program, and had one week to build it using a framework I'd never used before - I chose Vue.js because I liked its simiplicity and focus on component-based design.

Technologies Used: Vue.js, Express.js, MongoDB, Node.js, Bootstrap, Heroku, Netlify

This app is a personal food blog allowing users to create, read, update, or delete (CRUD) blog posts and data relating to their personal restaurant experiences. I created it in three days using Masonite (a new MVC framework for Python) for the Back End and React for the Front End.

Holiday Gifts App

This app allows users to create a list of gifts to buy during the holiday season, features user authentication & authorization, and displays helpful user-generated information about each gift. I created it using the Model-View-Controller architectural pattern, and the views components were coded in Liquid (the template language used to create all Shopify themes).

Technologies Used: Express.js, Node.js, Liquid, Mongoose, MongoDB, HTML, CSS, Javascript, Heroku
